Usability Test Results

Adding More Seal Imagery

The project's main focus is the seal itself, but there isn't any reference to it at all. Adding maybe an image or even a minimalistic vector illustration in the icon would be nice.

Adding a Drawing App

A user pointed out that a drawing app where users can freely draw whatever they want would be a good addition to the app. My target audience are 4th graders to 6th graders where if I'm just throwing information at them, they probably won't absorb any knowledge at all. Although I'm still considering about whether or not to implement it, a drawing app would add another playful element to the website to help retain attention span.

Letting Users Resize Windows

Everyone will have different sized screens, so adding the ability to help users even further customize their reading experience would help usability.
